# CodexUsageBar v0.1.1 Reliability Implementation Plan

> Design: `docs/superpowers/specs/2026-07-12-v0.1.1-reliability-design.md`

**Goal:** Add bilingual UI, actionable failure states, exact running-app discovery, privacy-safe diagnostics, and one authoritative version source without expanding the menu-bar-only product scope.

**Architecture:** Keep `UsageStore` responsible for refresh state and preserve structured failures. Add small support types for localization, executable resolution metadata, and application diagnostics. Keep AppKit interactions behind `AppSupport`, with `UsagePopoverView` limited to rendering and commands.

**Stack:** Swift 5.10, SwiftUI, AppKit, Swift Package Manager, XCTest, shell packaging scripts.
