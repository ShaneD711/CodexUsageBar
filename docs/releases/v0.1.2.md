# CodexUsageBar v0.1.2 - Unnotarized Preview

CodexUsageBar v0.1.2 is a reliability update for more accurate, predictable Codex usage reporting.

## Download

Download `CodexUsageBar-v0.1.2-macos-arm64.zip` and the matching `.sha256` file below.

## Requirements

- macOS 14 or later
- Apple Silicon Mac
- ChatGPT/Codex installed and signed in

## Highlights

- Detects signed-out accounts from structured account state
- Keeps one request deadline across initialization, account, and usage reads
- Reports stopped services immediately instead of waiting for a timeout
- Preserves the last successful usage snapshot when refresh fails
- Derives usage-window labels from their actual duration and order
- Uses clearer privacy-safe diagnostics without copying server messages
- Applies Simplified Chinese only to supported locales and follows the system region for reset dates and times
- Adds broader automated tests and GitHub Actions checks

## Security Notice

This preview is ad hoc signed, not signed with Apple Developer ID, and not notarized by Apple. macOS may block the first launch. Follow the README installation instructions and use `System Settings > Privacy & Security > Open Anyway` only if you trust this repository and the checksum.

## SHA-256

Verify the download with:

```bash
shasum -a 256 -c CodexUsageBar-v0.1.2-macos-arm64.zip.sha256
```
