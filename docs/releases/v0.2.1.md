# CodexUsageBar v0.2.1 - Unnotarized Preview

CodexUsageBar v0.2.1 focuses on one thing: keeping the remaining-usage display correct when Codex response structures change.

## Download

Download `CodexUsageBar-v0.2.1-macos-arm64.zip` and the matching `.sha256` file below.

## Requirements

- macOS 14 or later
- Apple Silicon Mac
- ChatGPT/Codex installed and signed in

## Highlights

- Supports the current weekly-only Codex usage response
- Preserves the last trusted usage values when a new response is malformed or ambiguous
- Distinguishes incompatible Codex versions, changed response formats, timeouts, sign-in state, and temporary service failures
- Validates cached usage before displaying it
- Handles unknown response fields and exact numeric representation changes conservatively
- Adds anonymized response fixtures and broader protocol, process, cache, and refresh tests

## Security Notice

This preview is ad hoc signed, not signed with Apple Developer ID, and not notarized by Apple. macOS may block the first launch. Follow the README installation instructions and use `System Settings > Privacy & Security > Open Anyway` only if you trust this repository and the checksum.

## SHA-256

Verify the download with:

```bash
shasum -a 256 -c CodexUsageBar-v0.2.1-macos-arm64.zip.sha256
```
