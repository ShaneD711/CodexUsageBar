# CodexUsageBar v0.2.0 - Unnotarized Preview

CodexUsageBar v0.2.0 makes the menu bar display more stable, compact, and predictable while keeping the app focused on one-glance Codex usage.

## Download

Download `CodexUsageBar-v0.2.0-macos-arm64.zip` and the matching `.sha256` file below.

## Requirements

- macOS 14 or later
- Apple Silicon Mac
- ChatGPT/Codex installed and signed in

## Highlights

- Adds Standard and Compact menu bar display modes
- Keeps percentage and reset-time widths stable during refreshes
- Shows `--` for unavailable values instead of presenting missing data as zero
- Preserves the last successful values and marks them when they become stale
- Uses clear availability states for loading, signed-out, missing-Codex, incompatible, and temporary failures
- Prevents older refresh results from overwriting newer usage data
- Improves menu bar tooltip, accessibility text, and privacy-safe diagnostics
- Adds stricter release validation for versioning, architecture, signing, extraction, and checksums

## Security Notice

This preview is ad hoc signed, not signed with Apple Developer ID, and not notarized by Apple. macOS may block the first launch. Follow the README installation instructions and use `System Settings > Privacy & Security > Open Anyway` only if you trust this repository and the checksum.

## SHA-256

Verify the download with:

```bash
shasum -a 256 -c CodexUsageBar-v0.2.0-macos-arm64.zip.sha256
```
